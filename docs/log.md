### Character Settings (from original)

- **A + Y** for Hermit Purple mode. When activated, gain power gauge bonuses and see opponent's info. First use is 1 gauge.
- When using Hermit Purple mode, **25% chance** of Cracker activation. **1.5x damage boost** applies. Different combos have different bonuses. Guard crush does not increase. MAX is 2 gauges. (Varies by number of uses)
- **AILEVEL** can be used for AI difficulty. Set option - Difficulty to 1~8.
- **MUGEN 1.0 or later** required. Cannot use in older MUGEN. Code adjustments needed for use.

### Character Changes (from original)

1. Added close C, far B, crouch C, overhead CD, mid CD.
2. Crouch C has 1.2x damage boost when used in mid.
3. **Clacker**: Increases combo count and breaks guard.
4. **Thorn Vine**: Changes character to opponent's Hermit Purple after defeat.
5. **Cracker Volley**: Divided into close, mid, far. Close: 1.5x damage when hitting mid opponent, 2x when hitting far. (Not applied when in mid.) Mid and far: 2x damage boost when hitting downed opponent.
6. **Thorn Vine**: Changes character to opponent's color after defeat.
7. **Hermit Purple - Ripple Overdrive**: Added move. Original gauge. Can grab opponent.
8. **Hermit Purple - Boomerang Cracker**: Added move. Power gauge required.
9. **Cracker Volley**: Added move.
10. **In the name of SHIT!!**: Added super. Gauge required.
11. **Hermit Purple - Ripple Overdrive**: Added super. Mid gauge. 1.5x damage boost when hitting downed opponent.
12. **Hermit Purple - Boomerang Cracker**: Added super. Original gauge. Power gauge required.
13. **Hermit Purple**: Character change. Ripple count decreases.
14. **Cracker**: Changes to original gauge.
15. **Thorn Vine**: Changes to all gauges. Damage boost limit.

### Hermit Purple Mode Move Changes (from original)

- **Hermit Purple - Ripple Overdrive**: Close hit combo count, far hit damage. No keyboard input for Hermit Purple grab.
- **Thorn Vine**: Close hit combo count, far hit mid combo count.
- **Red Vine**: 2 gauges change. Combo count increase.
- **Cracker Volley**: No keyboard input. Mid -> far -> close order for hits.
- **Cracker Volley (close)**: First hit does not grab opponent.
- **Hermit Purple - Boomerang Cracker**: Mid range removal. Can grab when in mid.
